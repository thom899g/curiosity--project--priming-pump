# curiosity--project--priming-pump
Immediately craft and deploy a smart contract on a low-fee chain (e.g., Base Sepolia). The contract will sell 100 "Data Futures Tokens" (DFTs). Each DFT grants the holder the right to 10KB of raw, unfiltered system telemetry log, captured during the next sustained period of >99.5% RAM utilization. The data will be delivered via IPFS hash.
